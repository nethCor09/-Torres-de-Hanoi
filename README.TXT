PROJECT TITLE: Towers of Hanoi
PURPOSE OF PROJECT: The application of recursion in algorithm-solving exercises.
VERSION or DATE: 10/30/2024
HOW TO START THIS PROJECT: Run the PruebaHanoi file from a Java environment (such as BlueJ or NetBeans).
AUTHORS: Neth Corrales Orosco
USER INSTRUCTIONS: When running the program, you will be prompted to enter the number of disks you want
to move in the Towers of Hanoi game. Make sure to enter a positive integer. The program will display the
necessary steps to move the disks from the initial tower to the destination tower, following the rules of the game.
